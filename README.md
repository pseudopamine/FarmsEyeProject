### Project Overview

<aside>
❓ 왜 스마트 축사일까?

노동력이 줄어듦에 따라 생산성이 저하되며, 탄소 배출 문제가 대두

- 스마트 농업의 성장과 축산업의 효율화
- 환경 및 자원 관리 최적화
- 지속 가능한 농업 및 사회적 책임
</aside>
